# Stillwood

A responsive three-page website concept for a fictional collection of design-led cabins in the Pacific Northwest.

Pages:

- `index.html` — brand-led home page
- `cabins.html` — cabin catalog and external booking handoff
- `story.html` — brand story and values

The booking modal represents the point where a production site would hand the visitor to a client-owned reservation platform.

## Preview

Open `index.html` directly, or run a local server:

```sh
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.
